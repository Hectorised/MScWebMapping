Maps Created Using Google API, Google Earth and KML

The Isle of Wight is located off the south coast of England. Within this webpage, a google api has been added to browse.

Attempts were made to add functionality to the javascript and overlay a KML or GeoJSON, but unfortunately this was in vain as neither the relative kml in the folder, nor one on google drive or github would render a layer. However, it provides a good template for the future.

Google earth and Geoserver Network Link

The geoserver networks link was exported and works successfully in Google Earth. the link is http://50cr018.sims.cranfield.ac.uk:8080/geoserver/dwallerwmmsc/wms?service=wms&request=GetMap&version=1.1.1&format=application/vnd.google-earth.kml+xml&layers=iow_roads&styles=iow_roads&height=452&width=768&transparent=false&srs=EPSG:4326
