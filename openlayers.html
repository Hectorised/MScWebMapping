Maps Created Using Geoserver and Open Layers

The Isle of Wight is Located of the south coast of England. Within this webpage are layers served from Geoserver into Open Layers

The base map has been created using an api from openlayers. Towns and the population are colour coded, as are highways by classificaiton. All layers were edited and styled in QGIS, then uploaded to geoserver. The styling was exported from QGIS as an SLD file. Regrettably I was unable to get the legend to load into the map in time.
