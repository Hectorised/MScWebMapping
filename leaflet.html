Leaflet API and QGIS2WEB

The Isle of Wight is located of the south coast of England. Within this webmap below is an API layer from Leaflet (https://leafletjs.com/SlavaUkraini/).

The javascript was more complex to understand in the time available therefore only the base as loaded, using code form Leaflet tutuorials

Within this webpage is a leaflet ouput from QGIS3Web.

The layers show a soilmap of the Isle of Wight. Pop-ups are enabled to look at the data. The file has been reproject to web mercator. Regrettably, I was unable to link this to series names as they were not supplied, so the codes are avaialable. Ironically, the base map would not export so the data is presented 'as is'.
